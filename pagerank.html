PENGERTIAN#

PageRank adalah algoritma yang digunakan oleh mesin pencari Google untuk menentukan urutan halaman web dalam hasil pencarian. Algoritma ini mengevaluasi jumlah dan kualitas tautan yang mengarah ke sebuah halaman untuk menentukan seberapa penting halaman tersebut. PageRank juga dapat digunakan untuk menganalisis jaringan sosial, serta jaringan kepercayaan atau peringkat kepercayaan pada perangkat lunak seperti email. Algoritma PageRank menghitung nilai penting (ranking) dari setiap node pada suatu grafik berbobot (weighted graph) dengan merangkai nilai penting dari node lain yang terhubung dengannya secara langsung dan tidak langsung.

RUMUS#

\(PR(A) = (1-d) + d \sum\limits_{i \in B_{A}} \frac{PR(i)}{L(i)}\)

di mana:

  • \(PR(A)\) adalah PageRank dari halaman A

  • \(d\) adalah faktor damping yang biasanya diatur ke 0,85

  • \(B_{A}\) adalah himpunan semua halaman yang memiliki tautan ke halaman A

  • \(L(i)\) adalah jumlah tautan keluar dari halaman i

  • \(PR(i)\) adalah PageRank dari halaman i

MEMBUAT FUNGSI readDoc#

  • Fungsi readDoc() bertujuan untuk membaca sebuah dokumen yang diinputkan pengguna melalui terminal, kemudian mengembalikan isi dari dokumen tersebut dalam bentuk string.

  • Fungsi ini pertama-tama meminta pengguna untuk memasukkan nama file. Selanjutnya, fungsi ini akan menentukan jenis file dokumen yang dimasukkan pengguna, apakah berformat .txt, .pdf, atau bukan keduanya. Jika dokumen berformat .txt, maka fungsi akan membuka file tersebut dan membaca isi dokumen ke dalam variabel document. Jika dokumen berformat .pdf, maka fungsi akan membuka file tersebut dan membaca teks pada halaman pertama ke dalam variabel document. Jika dokumen bukan berformat .txt atau .pdf, maka fungsi akan mengembalikan string kosong.

  • Fungsi ini menggunakan library PyPDF2 untuk membaca dokumen PDF, serta menggunakan built-in function open() pada Python untuk membaca dokumen teks.

def readDoc():
    name = input('Please input a file name: ') 
    print('You have asked for the document {}'.format(name))

    # now read the type of document
    if name.lower().endswith('.txt'):
        choice = 1
    elif name.lower().endswith('.pdf'):
        choice = 2
    else:
        choice = 3
        # print(name)
    print(choice)
    # Case 1: if it is a .txt file
        
    if choice == 1:
        f = open(name, 'r')
        document = f.read()
        f.close()
            
    # Case 2: if it is a .pdf file
    elif choice == 2:
        pdfFileObj = open(name, 'rb')
        pdfReader = PyPDF2.PdfFileReader(pdfFileObj)
        pageObj = pdfReader.getPage(0)
        document = pageObj.extractText()
        pdfFileObj.close()
    
    # Case 3: none of the format
    else:
        print('Failed to load a valid file')
        print('Returning an empty string')
        document = ''
    
    print(type(document))
    return document

MEMBUAT FUNGSI TOKENISASI#

  • Fungsi tokenize(document) digunakan untuk melakukan tokenisasi terhadap dokumen yang diberikan menjadi kalimat-kalimat terpisah.

  • Fungsi ini menggunakan library PunktSentenceTokenizer dari nltk.tokenize.punkt untuk melakukan tokenisasi. PunktSentenceTokenizer adalah sebuah kelas yang digunakan untuk membagi sebuah dokumen menjadi kalimat-kalimat berdasarkan aturan pemisahan yang sudah ditentukan sebelumnya.

  • Fungsi tokenize(document) menerima sebuah dokumen (dalam bentuk string) sebagai input, kemudian menggunakan PunktSentenceTokenizer untuk membagi dokumen tersebut menjadi kalimat-kalimat terpisah. Hasilnya, fungsi ini mengembalikan daftar (list) yang berisi setiap kalimat dari dokumen sebagai elemen-elemen dalam daftar tersebut.

def tokenize(document):
    # We are tokenizing using the PunktSentenceTokenizer
    # we call an instance of this class as sentence_tokenizer
    doc_tokenizer = PunktSentenceTokenizer()
    
    # tokenize() method: takes our document as input and returns a list of all the sentences in the document
    
    # sentences is a list containing each sentence of the document as an element
    sentences_list = doc_tokenizer.tokenize(document)
    return sentences_list

Demo pada countvectorizer#

digunakan untuk melakukan demo pada CountVectorizer dengan membuat objek CountVectorizer baru bernama cv_demo.

Kemudian, kita memiliki dua dokumen kecil dalam variabel text_demo. cv_demo.fit_transform(text_demo) digunakan untuk menghitung frekuensi kata dalam dokumen dan membuat matriks term-dokumen untuk kedua dokumen. Output matriks ini disimpan dalam variabel res_demo.

res_demo.toarray() menghasilkan matriks term-dokumen sebagai array numpy, dan cv_demo.get_feature_names_out() menghasilkan daftar fitur yang muncul dalam matriks term-dokumen.

Hasilnya menunjukkan bahwa kata-kata dalam kedua dokumen telah dihitung frekuensinya dan diwakili dalam matriks term-dokumen.

cv_demo = CountVectorizer()
text_demo = ["Ashish is good, you are bad", "I am not bad"] 
res_demo = cv_demo.fit_transform(text_demo)
print('Result demo array is {}'.format(res_demo.toarray()))

mengimplementasikan proses transformasi dari bag-of-words matrix menjadi normal matrix dengan menggunakan metode TF-IDF (Term Frequency-Inverse Document Frequency). Pada dasarnya, metode TF-IDF digunakan untuk memberi bobot pada setiap kata dalam setiap dokumen, di mana kata yang muncul lebih sering dalam dokumen tersebut akan memiliki bobot yang lebih besar, tetapi kata yang muncul di banyak dokumen akan memiliki bobot yang lebih kecil.

PageRank digunakan untuk memberi nilai pada setiap kalimat dalam dokumen berdasarkan kesamaannya dengan kalimat lain dalam dokumen tersebut. Semakin sering sebuah kalimat muncul dalam hubungan yang erat dengan kalimat lainnya, semakin tinggi nilai PageRank yang akan diterimanya.
